# The Factor Module

## Overview

<img src='inst/icons/analysis-classical-sem.svg' width='149' height='173' align='right'/>

**JASP Factor module** is a core module for JASP that provides tools for exploring hidden structure in the data. The Factor module offers principal component analysis, exploratory factor analysis, and confirmatory factor analysis.


## R Packages

<img src='https://www.r-project.org/logo/Rlogo.svg' width='100' height='78' align='right'/>

The functionality is served by several R packages

- **lavaan** — Latent variable analysis ([lavaan on CRAN](https://cran.r-project.org/package=lavaan))
- **psych** — Procedures for psychological, psychometric, and personality research ([psych on CRAN](https://cran.r-project.org/package=psych))
- **qgraph** — Graph plotting methods and psychometric data visualization ([qgraph on CRAN](https://cran.r-project.org/package=qgraph))
- **GPArotation** — Gradient projection algorithm rotation for factor analysis ([GPArotation on CRAN](https://cran.r-project.org/package=GPArotation))
- **semPlot** — Path diagrams and visual analysis of structural equation models ([semPlot on CRAN](https://cran.r-project.org/package=semPlot))
